# ProTasker API

A simple Task Management REST API built with FastAPI and SQLite.

## Features

- Create a task
- Get all tasks
- Get a task by ID
- Update a task
- Delete a task
- SQLite database for permanent storage

## Technologies Used

- Python
- FastAPI
- SQLite
- Pydantic
- Uvicorn

## Installation

```bash
pip install -r requirements.txt
```

## Run the project

```bash
python -m uvicorn main:app --reload
```

## API Documentation

Open your browser and visit:

```
http://127.0.0.1:8000/docs
```
